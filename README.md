# District Care

Мобільна платформа для небайдужих мешканців: швидко зафіксувати екологічне порушення, позначити на карті і передати сигнал далі.

**Географія:** Вся Україна · **Платформа:** Mobile-first веб · **Мова:** Українська

## Ключова механіка

Швидка фіксація порушення (3–4 дії: геолокація + фото + категорія + відправка) → модерація → відображення на карті.

Фреймінг: не скарга, а **сигнал**. Користувач робить вклад — не чекає реакції.
